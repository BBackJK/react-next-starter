# react-next-starter

react hooks and next framework stater kit with babel, eslint.

## git clone

```
git clone https://github.com/BBackJK/react-next-starter.git
cd react-next-starter/
npm install
```

## start

- development next cli

```
npm run dev     // next
```

- build next cli

```
npm run build   // next build
```

- start next cli

```
npm run start
```
